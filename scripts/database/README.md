# Database Scripts

This folder contains scripts for database connectivity testing and diagnostics.

## Scripts

- `check_supabase_connection.py` - Comprehensive Supabase database connection diagnostic
- `check_admin_user.py` - Admin user verification and setup script

## Usage

```bash
# Test Supabase connection
python scripts/database/check_supabase_connection.py

# Check admin user setup
python scripts/database/check_admin_user.py
```

## Purpose

These scripts help with:
- Database connectivity verification
- Configuration validation
- Troubleshooting connection issues
- Admin user management
