# Shell Scripts

This folder contains utility shell scripts for development and deployment.

## Scripts

- `run_local.sh` - Run the application locally with proper environment setup
- `setup_local_dev.sh` - Set up local development environment

## Usage

```bash
# Run application locally
./scripts/shell/run_local.sh

# Setup local development environment
./scripts/shell/setup_local_dev.sh
```

## Purpose

These scripts automate:
- Local development server startup
- Environment variable configuration
- Dependency installation
- Database setup and migrations
